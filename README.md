# 🎬 Movie Ticket Booking System

A comprehensive web-based platform designed to streamline the movie ticket booking process. This system allows users to browse movies, select showtimes, and book seats, while providing administrators with tools to manage theater inventory, show schedules, and bookings.

## 🚀 Key Features

*   **User Module:**
    *   Browse "Now Showing" and "Coming Soon" movies.
    *   Interactive seat selection interface.
    *   User authentication and profile management.
    *   Real-time booking history and ticket confirmation.
*   **Admin Module:**
    *   Manage movie listings (add, update, delete).
    *   Configure theater screens and seating layouts.
    *   Schedule showtimes for specific screens.
    *   Dashboard for viewing booking analytics and performance.

## 🛠 Tech Stack

*   **Frontend:** [e.g., React.js, Tailwind CSS]
*   **Backend:** [e.g., Node.js, Express.js]
*   **Database:** [e.g., MongoDB, PostgreSQL]
*   **Authentication:** [e.g., JWT, Firebase Auth]

## 📋 Prerequisites

Before you begin, ensure you have the following installed on your machine:

*   Node.js (vXX.X.X or higher)
*   [e.g., MongoDB]
*   npm or yarn

## ⚙️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/muhammadaminusmonov/movie_ticket_system.git
    cd movie_ticket_system
    ```

2.  **Install dependencies:**
    ```bash
    # Navigate to server and install
    cd server
    npm install

    # Navigate to client and install
    cd ../client
    npm install
    ```

3.  **Environment Variables:**
    Create a `.env` file in the `server` directory and add the following:
    ```env
    PORT=5000
    MONGO_URI=your_mongodb_connection_string
    JWT_SECRET=your_secret_key
    ```

4.  **Run the application:**
    ```bash
    # Start the server
    cd server
    npm run dev

    # Start the client (in a new terminal)
    cd client
    npm start
    ```
